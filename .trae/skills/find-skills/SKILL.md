# Find Skills

This skill helps users discover and find available skills in the workspace.

## When to Use

- User asks to find skills
- User wants to list available skills
- User asks "what skills are available"
- User wants to discover skills

## How to Use

1. Search for skills in the `.trae/skills/` directory
2. List all available skills with their descriptions
3. Provide information about how to use specific skills

## Available Skills

Skills are located in `.trae/skills/` directory. Each skill has its own subdirectory with a `SKILL.md` file.

## Examples

- "Find available skills"
- "List all skills"
- "What skills can I use?"
- "Show me the skills"
